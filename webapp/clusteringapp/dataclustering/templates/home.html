Clustering

Data Clustering is the process of organizing objects into groups whose members are similar in some way.

We have created this website as a tool for people to use who are looking to run clustering algorithms on their own data. The intention of this website is to be continuously evolving and expanding in regards to the distance metrics we offer, different display options, and other various projects we are working on implementing.

kluster.me

This application was originally written as a project for a class in bioinformatics. We utilized many different tools for this project including: Python for our clustering algorithm. HTML and CSS for creating and styling our website. Javascript, Django, Heroku, and MySQLite for back end and serverside. We learned a lot throughout the semester and hopefully plan on continuing working on this project in our free time as a learning tool.
